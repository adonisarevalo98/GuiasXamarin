SEGUNDO PROYECTO.

- Introducción al modelo MVVM.

El proyecto pretende:
1. Familiarizarse con la arquitectura de carpetas y flujo de datos.
2. Uso de Bindings para mostrar datos con el flujo: Model -> ViewModel -> View
2. Creación de formularios y validación (Simple) de entradas.
3. Listar datos haciedo uso de CollectionView.
4. Hacer uso de SQLite para Guardar y desplegar datos en una pantalla (Con MVVM).
